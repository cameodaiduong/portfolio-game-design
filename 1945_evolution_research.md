# 1945 Air Force — Evolution Research & Game Analysis

## Bối cảnh công ty

**Onesoft** thành lập 01/04/2010, bởi 4 người bạn → sau tách thành 4 studio:
- **Falcon Game Studio** (1945 Air Force, Galaxiga, Screw Puzzle)
- Rocket Game Studio
- ABI Game Studio  
- Monster Game Studio

Nhân sự tăng 100x trong 12 năm. Leading position trong global mobile game market (cả dev lẫn publishing).

## Timeline phát triển 1945 Air Force

### Phase 1: Launch & Core Loop (03/2019)
**Release:** 15/03/2019 trên iOS & Android

**Core game ban đầu (suy luận từ dữ liệu):**
- Vertical scrolling shoot 'em up, lấy cảm hứng từ classic 1942/1943/1944
- Core mechanic: auto-shoot + drag to dodge
- Campaign mode với các level tuần tự
- Hệ thống plane cơ bản (6 loại WWII planes ban đầu)
- Monetization đơn giản: ads + IAP cơ bản

**Tại sao thành công ngay?**
- Nostalgia factor cực mạnh — thế hệ 8x/9x đều biết 1942/1943
- Market gap: ít game shmup chất lượng trên mobile thời điểm đó
- Simple core mechanic = low barrier to entry
- Falcon đã có kinh nghiệm từ Falcon Squad (game trước đó cùng genre)

### Phase 2: Content Expansion & Retention Layer (2019-2020)
**Dấu hiệu:**
- Từ 6 planes → 60+ planes (content explosion)
- Thêm **Wingman system** — máy bay phụ bay cùng
- Thêm **Device system** — equipment cho plane
- Concept **Loadout** = Aircraft + Wingmen + Devices
- Campaign mở rộng liên tục (hiện 900+ levels)

**Tại sao làm điều này?**
- Core loop đơn (bắn → qua level) cạn dopamine sau ~7-14 ngày
- Cần **meta-game layer** để giữ player dài hạn
- Collection mechanic (sưu tập planes) = long-term goal
- Upgrade depth (plane + wingman + device) = decision complexity tăng dần
- **Revenue insight:** Dec 2020 đạt $800K/tháng revenue, 4M downloads/tháng

### Phase 3: Social & Competitive (2020-2021)
**Features thêm:**
- **PvP real-time** — player vs player
- **Clan system** — tạo/join squadron
- **Global leaderboard**
- **Team-based events**

**Tại sao làm điều này?**
- Single-player content = finite. Multiplayer = infinite replayability
- Social obligation (clan duty) = retention hook mạnh nhất trong mobile gaming
- PvP tạo **competitive dopamine** — khác với PvE dopamine
- Leaderboard = status/ego → whale spending trigger
- UA milestone: #2 install volume ANZ & SEA (2020), 1M+ DAU

### Phase 4: LiveOps & Economy Deepening (2021-2023)
**Features thêm:**
- **Tournament system** (1 week, tiered)
- **Special Events** (rotate 2 tuần/lần)
- **Tier system** cho devices (tier 2, 3)
- **Aircraft Rating System**
- **Level system**
- Liên tục thêm 30-40 levels mỗi update

**Tại sao làm điều này?**
- Game mature → cần LiveOps để giữ existing players
- Events = FOMO mechanic → daily login
- Tiering system = power creep có kiểm soát → whale spending sustained
- Rating system = matchmaking quality → reduce churn from unfair PvP
- Google công nhận: Game of the Year 2019, Top 4 Developer VN 2023

### Phase 5: Platform Expansion & New Systems (2023-2026)
**Features thêm:**
- **Google Play Games on PC** — game VN đầu tiên ở SEA vào category này
- **Webshop** (1945.falcongames.com) — bán direct, bypass 30% store fee
- **Daily Check-in system**
- **Engine System** (mới nhất, v15.52)
- **Aircraft Skin system** (mới nhất)
- **Lucky Roll event** (gacha-style)
- **40 new levels** mỗi update gần đây

**Tại sao làm điều này?**
- PC expansion = new audience without new game
- Webshop = margin optimization (tránh 30% Google/Apple cut)
- Skin system = cosmetic monetization (không P2W) → healthier economy
- Gacha (Lucky Roll) = high ARPU mechanic
- Engine system = new upgrade dimension khi plane/wingman/device đã saturated

## Phân tích Pattern: Tại sao 1945 sống 7 năm?

### 1. Layered Complexity Model
```
Launch:     [Core Shoot] 
+6 months:  [Core Shoot] + [Upgrade Tree]
+12 months: [Core Shoot] + [Upgrade Tree] + [Collection]
+18 months: [Core Shoot] + [Upgrade Tree] + [Collection] + [PvP]
+24 months: [Core Shoot] + [Upgrade Tree] + [Collection] + [PvP] + [Social]
+36 months: [Core Shoot] + [Upgrade Tree] + [Collection] + [PvP] + [Social] + [LiveOps]
+60 months: ... + [Webshop] + [Skins] + [Gacha] + [New Systems]
```

Không ship hết cùng lúc. Mỗi layer thêm vào khi layer trước bắt đầu flatten retention curve.

### 2. Monetization Evolution
```
Phase 1: Ads only (low ARPU, high volume)
Phase 2: Ads + IAP planes (mid ARPU)
Phase 3: Ads + IAP + PvP advantage (whale targeting)
Phase 4: Ads + IAP + Events + VIP (subscription-like)
Phase 5: Ads + IAP + Webshop + Skins + Gacha (full spectrum)
```

### 3. Đối thủ cùng genre & so sánh
- **Sky Force Reloaded** — premium quality nhưng paid model → limited reach
- **HAWK: Freedom Squadron** — My.com, aggressive P2W → bad reputation
- **Galaga Wars** — Bandai Namco, IP mạnh nhưng content thin
- **Space Shooter: Galaxy Attack** — clone trực tiếp, lower quality

1945 thắng vì: **free entry + deep enough meta + consistent content update + không quá P2W (so với HAWK)**

## Key Insights cho GDD

1. **Level design là engine of retention** — không phải features. Features giữ người quay lại, nhưng level design quyết định session length và satisfaction
2. **Difficulty curve = dopamine curve** — sawtooth pattern, không linear
3. **Mỗi system mới = new reason to grind** — khi plane upgrade max, thêm wingman. Khi wingman max, thêm device. Khi device max, thêm engine. Infinite treadmill.
4. **Social ≠ chat** — social trong 1945 = obligation (clan duty) + competition (PvP rank). Không cần chat system phức tạp.
5. **Monetization follows engagement, not vice versa** — họ không design game quanh monetization. Họ design engagement trước, rồi monetize engagement đó.
