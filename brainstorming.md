## 1. Falcon Game Studio — Đáng đi không?

**Tín hiệu tích cực:**
- Revenue ~$35M/năm, ~200 nhân sự, 9 năm hoạt động (2017-2026) — công ty stable, không phải startup cháy tiền
- Top 4 Game Developer VN (Google 2023), Game of the Year 2019 (1945)
- 5+ tỷ lượt tải, 600+ tựa game — họ biết publish & scale
- Lương intern 15M net — cao hơn trung bình thị trường đáng kể
- Benefits rất mạnh: 18 tháng lương, budget học 20M/năm, training từ chuyên gia Google/Meta
- JD viết rõ ràng: 80% level design, 20% UX/data — không mơ hồ

**Tín hiệu cần lưu ý:**
- Chủ yếu làm **hyper-casual / hybrid-casual** — không phải AAA hay mid-core sâu. Nếu muốn học deep game design (narrative, system design phức tạp), đây không phải nơi tốt nhất
- Mô hình kinh doanh nặng về **publishing + ads monetization** (AdMob, Meta, TikTok ads) — nghĩa là focus nhiều vào metrics, retention, không phải "nghệ thuật game"
- Review ITviec bị 403 nên không đọc được chi tiết, nhưng search thấy mention "flexible time, good comp, open culture"
- Yêu cầu **6 ca/tuần** cho intern — khá demanding

**Kết luận:** Đáng đi. Đặc biệt cho người chuyển ngành vì:
1. Học được data-driven design (rất marketable)
2. Ship game thật, có metrics thật
3. Lương intern tốt, không bị bóc lột
4. Puzzle game level design = entry point tốt cho game design career

## 3. Kế hoạch GDD 1945 Air Force (Deadline: 25/06)

GDD phân tích cần cover:

- **Core Loop:** Tap to dodge + auto-shoot → kill enemies → earn coins/gems → upgrade planes → harder levels
- **Progression Systems:** Campaign 900+ levels, plane unlock/upgrade tree, parts system
- **Monetization:** Dual currency (coins/gems), ads for continue/boost, IAP planes ($$$), VIP
- **Retention Hooks:** Daily quests, PvP, clan, events, boss fights
- **Strengths:** Nostalgia factor, simple core mechanic nhưng deep upgrade path, bullet-hell satisfaction
- **Weaknesses:** Aggressive monetization wall, P2W perception trong PvP

**Thế mạnh khi viết GDD này:** SWE background → hiểu data flow, system architecture. Viết GDD dưới góc nhìn "systems thinker" — phân tích economy loop bằng spreadsheet, vẽ state diagram cho progression. Đây là thứ intern thuần game design KHÔNG làm được.

## 4. Portfolio Game (Deadline: 26/06)

**Đề xuất:** Thiết kế 1 **puzzle game nhỏ** (match JD — họ cần puzzle designer)

Hướng gợi ý:
- **Screw Puzzle variant** — họ đang có Screw Puzzle, design 1 concept tương tự nhưng twist mới → chứng minh hiểu product line của họ
- Hoặc **hybrid puzzle-shooter** — blend 1945 DNA + puzzle mechanics → creative, memorable
- GDD ngắn gọn: 5-10 trang, focus core mechanic + 5 level design samples + monetization plan

**Khai thác thế mạnh SWE:**
- Có thể prototype bằng code (nếu kịp) — interactive demo >> static document
- System design diagram cho economy
- Data model cho level difficulty curve

## 5. CV Strategy cho người chuyển ngành

**Frame:** "Systems Engineer → Systems Designer"
- SWE 3-4 năm = hiểu how systems work, data analysis, user behavior
- Translate: "built APIs" → "designed user-facing systems", "optimized performance" → "balanced system parameters"
- Section riêng cho transferable skills: logic, data analysis, prototyping, technical communication
- Portfolio links nổi bật (GDD + game design) ngay đầu CV
- Lưu ý JD nói: **"1 file đính kèm duy nhất, portfolio dùng hyperlink/QR trong CV"**
